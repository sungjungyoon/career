## Structure

- `src/content/career.mdx` — narrative resume
- `src/content/projects/*.mdx` — one file per project (frontmatter + long-form body)

## Why MDX

Markdown is great for prose; JSON is rigid for long text. MDX sits in the middle: structured metadata **and** rich content in one place.
